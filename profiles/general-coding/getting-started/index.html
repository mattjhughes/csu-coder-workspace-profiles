Getting Started: General Coding

This workspace is tuned for standard software development. Use VS Code for editing, File Browser for quick uploads and downloads, and the terminal for running tools like codex.

VS Code

Open the editor to inspect the repo, change files, and use the integrated terminal for most development work.

File Browser

Use it to upload source files, download outputs, and move files around without leaving the browser.

Terminal

Run codex directly in the terminal when you want the agent to inspect, edit, and validate code for you.

Best for

  • backend services
  • APIs and scripts
  • small frontend changes in existing apps
  • general engineering support

Good first prompt

“Inspect this repo and tell me the best way to implement the feature I describe next. Then make the change and validate it.”
