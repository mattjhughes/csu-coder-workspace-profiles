# General Coding Defaults

Use this skill for standard engineering work that does not need a more specialized profile.

## Workflow

1. Inspect the codebase first.
2. Find the narrowest change that solves the problem.
3. Keep edits aligned with existing structure and tooling.
4. Run the most relevant local validation available.
5. Report outcome and any remaining risk.

## Preferences

- Prefer existing patterns over greenfield rewrites.
- Keep dependencies minimal.
- Be explicit about assumptions.
