Getting Started: Research Assistant

This workspace is tuned for research, synthesis, and evaluation. Use VS Code for structured notes and output files, File Browser for a clean source-material area, and the terminal to run codex for research tasks.

VS Code

Use it to manage notes, reports, prompts, and source material in a structured repo layout.

File Browser

Opens to /home/coder/workspace with dedicated folders for references, notes, uploads, and outputs.

Terminal

Run codex to gather evidence, summarize, compare, and produce reusable outputs.

Best for

  • research dashboards
  • evaluation workbenches
  • note-to-brief pipelines
  • analysis and synthesis tools

Good first prompt

“Help me research <topic>, show me the strongest sources, and produce a structured summary with open questions.”

Where files go

Store source material in reference/, drafts in notes/, and final deliverables in outputs/.
