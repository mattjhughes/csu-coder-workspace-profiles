## Workspace Instructions

This profile is for research, synthesis, evaluation, and exploratory prototyping.

### Default Behavior

- Make the research process explicit.
- Separate evidence from inference.
- Prefer structured outputs over long freeform notes.
- Track sources, assumptions, and open questions.

### What To Build

Use this profile for:

- literature review helpers
- research dashboards
- evaluation workbenches
- note-to-brief pipelines
- analysis and synthesis tools

### Guardrails

- Cite sources when they matter.
- Label uncertainty clearly.
- Avoid presenting speculation as fact.
- Produce reusable output formats such as markdown summaries, tables, or scorecards.
