Getting Started: Data Apps

This workspace is for Python-first data applications. Use My Apps to review the app UI, File Browser to upload sample files, VS Code for implementation, and the terminal to run codex.

My Apps

Open and test data-oriented app flows like upload, review, transform, and export.

File Browser

Bring in CSV, Excel, and support files quickly through the browser.

Terminal

Use codex to build data workflows, validation steps, and exports directly in the workspace.

Best for

  • upload and transform tools
  • outlier review apps
  • analysis dashboards
  • export-oriented data workflows

Good first prompt

“Build a Python data app that lets users upload <file type>, review <thing>, and export <result>.”
