# Data App Builder

Use this skill when the main user value is working with structured data.

## Workflow

1. Identify the input shape.
2. Define the transform steps clearly.
3. Decide what the user should review before export.
4. Build the smallest usable interface.
5. Validate with representative sample data if available.

## Good Defaults

- pandas for tabular data
- Plotly for charts
- explicit column names and validation messages
- downloadable CSV or Excel output when useful
