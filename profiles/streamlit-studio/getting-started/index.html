Getting Started: Streamlit Studio

This workspace is designed for dashboards and internal data apps. Use My Apps to open the user-facing interface, VS Code to edit the code, File Browser for assets and uploads, and the terminal to run codex.

My Apps

Open the Streamlit app studio and test the pages users will actually see.

VS Code

Edit app files, dependencies, and starter pages without leaving the browser.

File Browser

Quickly upload CSVs, download artifacts, and manage files in the workspace.

Best for

  • KPI dashboards
  • CSV and Excel explorers
  • reporting portals
  • review and approval workflows

Good first prompt

“Build a Streamlit app for me. The audience is <who>, the data source is <what>, and users need to <top actions>.”
