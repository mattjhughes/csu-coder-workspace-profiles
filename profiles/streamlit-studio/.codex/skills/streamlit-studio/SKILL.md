# Streamlit Studio

Use this skill when the user wants a dashboard, data app, or lightweight internal tool.

## Workflow

1. Pick a simple app shape.
2. Build in Python with Streamlit.
3. Use Plotly for charts when it improves readability.
4. Keep pages short and scannable.
5. Validate health and visual quality after changes.

## Prompting Guidance

If the user is vague, ask for:

- the data source
- the audience
- the top 1-3 actions users need
- any must-have charts or exports
